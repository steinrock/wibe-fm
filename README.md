# wibe-fm
Простой файловый менеджер на базе PHP для сервера, реализованный с помощью wibe-coding через Gemini для личного использования
Реализованы функции загрузки и скачивания файлов и каталогов, перемещения и удаления файлов, а также загрузки файлов группой или в составе директории с сохранением иерархии
Этот файловый менеджер был написан нейросетью Gemini, протестирован и отлажен мной. В серьезных задачах применять с осторожностью, так как неизвестно, какие дыры может оставить за собой нейросеть. Создал я его для себя, чтобы упростить работу с файлами на своем сервере, но считаю, что этот продукт потенциально может быть полезен и для вас.
